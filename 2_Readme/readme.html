Purpose & Aim

Super Tracker records statistics for each user arriving at your shop and allows you to generate various reports based on this data.

Super Tracker records the following data:

  • Referring page
  • Referring query allows access to keywords used in search engine)
  • Landing page (including query string - important for Pay Per Click campaign assessment)
  • Arrival time / exit time /exit page
  • IP address
  • Country code / country name
  • Customer's cart contents
  • Number of clicks on shop (accuracy is not 100% as this is recorded based on session ID)
  • Categories Viewed
  • Products viewed
  • Customer ID (if signed in)
  • Order ID (if order was placed)
  • Were products added to cart (true/false)
  • Did the customer Checkout (true/false)

Super Tracker then uses this data to generate many interesting reports about customer activity on your site, without the need of a 3rd party service. Helps merchants answer questions like:

  • How do my customers find my site?
  • Which referring sites generate the most sales?
  • Are there any trends which may help identify why customers are leaving without completing their purchase?
  • Am I getting enough visitors from the countries I am targeting?
  • Is my Pay Per Click Campaign delivering pay customers, rather than just expensive hits?

Reports Generated:

  • Top Referrers
  • Top Sales Generating Referrers
  • Visiting Countries Stats
  • Search Keywords Used - Totals
  • Search Keywords Used in Last 24 Hours
  • Search Keywords Used in Last 3 Days
  • Search Keywords Used in Last Week
  • Search Keywords Used in Last Month
  • Top Exit Pages (No Sale)
  • Top Exit Pages (No Sale, but added to cart)
  • Average Clicks on Site by Referrer
  • Average Time Spent on Site by Referrer
  • Products Viewed Report
  • Last Ten Visitors

Installation

!!!!!BACKUP YOUR FILES AND DATABASE BEFORE YOU BEGIN!!!!!

BEFORE INSTALLING, you are strongly encouraged to READ ALL OF THE TABS in this readme file. Then follow the installation or upgrade instructions, as appropriate.

BEFORE YOU POST A QUESTION ON THE FORUM - read through this readme again as needed

There are two folders in the fileset

  • 1_Install_Files
  • 3_files_for_zen_cart_1.3.x
New Installation Instructions
  1. BACK UP your database & store.
  2. Unzip the Super Tracker package to your local hard drive, retaining the folder structure.
  3. Rename the "YOUR_ADMIN_FOLDER" folder in the "1_Install_Files"
    • 1_Install_Files/YOUR_ADMIN/
  4. Rename all of the "YOUR_TEMPLATE" folders in the "1_Install_Files" folders to match the name of your template folder.
    • 1_Install_Files/includes/templates/YOUR_TEMPLATE/
  5. Upload the files from "1_Install_Files" to your store. (DO NOT upload the "1_Install_Files" folder, just the CONTENTS of this folder -
    copy ALL of the add-on files to your store!! Most issues are caused by store owners who decide to NOT load ALL of the module files)
  6. Download the latest GeoIP.dat file from MaxMind or direct download the zipped file. This file is updated every month.
  7. Unzip the file and FTP it to your store "includes/GeoIP.dat"
  8. Once the files are uploaded, click ANY link in your store's admin, and the auto-installer will be triggered. If you have followed the install instructions, you will see the installation confirmation message.
  9. Optional: If you are still using Zen cart 1.3.x also also upload from the "3_files_for_zen_cart_1.3.x" folder:
    • YOUR_ADMIN/includes/boxes/extra_boxes/supertracker_reports_dhtml.php
  10. You are ready now to start configuring Super Tracker
Upgrade Instructions

If you are upgrading from ANY older version of Super Tracker (ALL PRIOR VERSIONS!!).

  1. BACK UP your database & store.
  2. Unzip the Super Tracker package to your local hard drive, retaining the folder structure.
  3. Rename the "YOUR_ADMIN_FOLDER" folder in the "1_Install_Files"
    • 1_Install_Files/YOUR_ADMIN/
  4. Rename all of the "YOUR_TEMPLATE" folders in the "1_Install_Files" folders to match the name of your template folder.
    • 1_Install_Files/includes/templates/YOUR_TEMPLATE/
  5. Upload the files from "1_Install_Files" to your store. (DO NOT upload the "1_Install_Files" folder, just the CONTENTS of this folder -
    copy ALL of the add-on files to your store!! Most issues are caused by store owners who decide to NOT load ALL of the module files)
    Overwrite all the older files from Super Tracker.
  6. Download the latest GeoIP.dat file from MaxMind or direct download the zipped file. This file is updated every month this installation contains the file generated on 8-8-2012
  7. Unzip the file and FTP it to your store "includes/GeoIP.dat"
  8. Once the files are uploaded, click ANY link in your store's admin, and the auto-installer will be triggered to update your database. If you have followed the install instructions, you will see the installation confirmation message.
  9. If you are using Zen cart 1.5.x delete:
    • YOUR_ADMIN/includes/boxes/extra_boxes/supertracker_reports_dhtml.php
    This file is now obsolete
Un-Install Instructions
  1. BACK UP your database & store.
  2. Run the uninstall.sql from the "Sql files" folder
  3. Delete the following from your current install:
    • YOUR_ADMIN/supertracker.php
    • YOUR_ADMIN/images/flags/ (complete directory)
    • YOUR_ADMIN/includes/searchengines.php
    • YOUR_ADMIN/includes/auto_loaders/config.supertracker.php
    • YOUR_ADMIN/includes/extra_datafiles/supertracker.php
    • YOUR_ADMIN/includes/functions/extra_functions/supertracker.php
    • YOUR_ADMIN/includes/init_includes/init_supertracker_config.php
    • YOUR_ADMIN/includes/installers/supertracker/ (complete directory)
    • YOUR_ADMIN/includes/languages/YOUR_LANGUAGE/supertracker.php
    • YOUR_ADMIN/includes/languages/YOUR_LANGUAGE/extra_definitions/supertracker.php
    • includes/GeoIP.dat
    • includes/geoip.inc
    • includes/classes/supertracker.php
    • includes/extra_datafiles/supertracker.php
    • includes/templates/YOUR_TEMPLATE/jscript/jscript_supertracker.php
  4. If you are still using Zen cart 1.3.x also delete:
    • YOUR_ADMIN/includes/boxes/extra_boxes/supertracker_reports_dhtml.php

Configuration

  1. Go to Configuration => Super Tracker Configuration in the admin panel.
    • Enter a Comma Separate List of IP's which should not be recorded, for instance, your own PC's IP address, or that of your server if you are using Cron Jobs, etc
    • Enter a Comma Separate List of UserAgent's substring which should not be recorded
    • Choose to exclude, or include the recording of bots
  2. That's, all enjoy

Usage

Reports

You can view the reports Super Tracker is generating via Admin => Reports Super Tracker

Super Tracker generates the following reports:

  • Top Referrers
  • Top Sales Generating Referrers
  • Visitors Countries Stats
  • Search Keywords Used - Total
  • Search Keywords Used in last 24h
  • Search Keywords Used in last 3 Days
  • Search Keywords Used in last Week
  • Search Keywords Used in last Month
  • Top Exit Pages (No Sale)
  • Top Exit Pages (No Sale, but added to cart)
  • Average Clicks on Site by Referrer
  • Average Time Spent on Site by Referrer
  • Products Viewed Report
  • Last Ten Visitors

Click on an item form the drop-down and view the report.

If you think there are to many records kept in the database you can erase the oldest ones. But keep in mind this will also influence the generated reports, since the deleted records information is no longer available.

Troubleshooting

Basics

Make sure your custom template is active. (Admin > Tools > Template Selection)

Make sure Super Tracker is installed. Admin => Configuration => Super Tracker Configuration

If Super Tracker does not work or gives you errors:

  • Make sure all files are in correct location
  • Make sure you uploaded ALL the Super Tracker files
  • Make sure the files are not corrupt from bad FTP transfers
  • MAKE SURE YOU RE-READ THE CONFIGURATION AND USAGE SECTIONS!!!
